# Azure Cloud Labs

This repository documents my hands-on journey through Microsoft Azure, covering core cloud infrastructure and data platform services.

The focus is on practical implementation, system behavior, and real-world troubleshooting—not just completing labs.

---

## 🔧 Areas Covered

- Compute (Virtual Machines, CLI deployments)
- Networking (VNets, NSGs, connectivity)
- Storage (Blob, File Shares)
- App Services & APIs
- Containers (ACI, container-based deployments)
- Monitoring & diagnostics
- Azure SQL Database
- Data pipelines using Azure Data Factory
- Infrastructure as Code (Terraform, CLI)
